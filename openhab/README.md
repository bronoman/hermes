# openHAB skill for Hermes Agent

## name: openhab

## description
Control and monitor zour local openHAB (www.openhab.org) smart home server using Hermes Agent.
Manage the following openHAB objects via REST API (see openHAB documentation for details) 
- things/devices
- items (e.g. single sensor on a device)
- rules

## add-on features
- health check for server, gateways and across all objects
- device status, battery levels, gateway monitoring
- 20 sec system diagnostics via log file monitoring
